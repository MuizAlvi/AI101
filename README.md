# AI101
Repository containing hands-on, open-source, practise notebooks for participants attending the [AI 101: Introduction to Artificial Intelligence Workshop](https://www.facebook.com/events/429419051422702). 

List of files:

- For Machine Learning:
  - 01 - Data Exploration
  - 02 - Regression
  - 03 - Classification
  - 04 - Clustering

- For Deep Learning:
  - 05a - Deep Neural Networks (PyTorch)
  - 05a - Deep Neural Networks (TensorFlow)
  - 05b - Convolutional Neural Networks (PyTorch)
  - 05b - Convolutional Neural Networks (Tensorflow)
  - 05c - Transfer Learning (PyTorch)
  - 05c - Transfer Learning (Tensorflow)
